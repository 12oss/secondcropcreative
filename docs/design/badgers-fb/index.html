Wisconsin Badgers Football Watercolor

Vibrant colors and expressive, watery strokes have always been common for watercolor artists. Water and gravity work together, in a consensual relationship, to create an organic yet abstract blooms and drips on paper. I'm too OCD to allow that to happen, so I've taken to the digital canvas to create very controlled water color renditions in photoshop. This Wisconsin Badger Football painting was my first attempt at a new process (for me).
